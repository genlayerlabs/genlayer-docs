# GenLayer Simulator

The GenLayer Simulator replicates the full GenLayer network, including validators and the execution environment, allowing you to test and interact with Intelligent Contracts in a controlled setting.

## What you can do with the GenLayer Simulator

- **Experiment with AI Smart Contracts**: Build Intelligent Contracts that can process natural language using LLMs such as GPT-4, Llama3, OpenAI, etc.
- **Analyze Web Data**: Build Intelligent Contracts that can access and analyze web data.
- **Build with Python**: Develop in a familiar, developer-friendly language.
- **Test in a Sandbox**: Simulate the GenLayer network, including validators and the execution environment.
