---
title: staking wizard
---

Interactive wizard to become a validator

### Usage

`$ genlayer staking wizard [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --account &lt;name&gt;             Account to use (skip selection) | No |  |
|  |  | --network &lt;network&gt;          Network to use (skip selection) | No |  |
|  | --skip-identity | Skip identity setup step | No |  |
|  |  | --rpc &lt;rpcUrl&gt;               RPC URL for the network | No |  |
|  |  | --staking-address &lt;address&gt;  Staking contract address (overrides chain config) | No |  |
| -h | --help | display help for command | No |  |
