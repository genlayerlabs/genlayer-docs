---
title: staking validator-claim
---

Claim validator withdrawals after unbonding period

### Usage

`$ genlayer staking validator-claim [options] [validator]`

### Arguments

- `[validator]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  | --validator &lt;address&gt; | Validator wallet contract address (deprecated, use positional arg) | No |  |
|  | --account &lt;name&gt; | Account to use | No |  |
|  | --password &lt;password&gt; | Password to unlock account (skips interactive prompt) | No |  |
|  | --network &lt;network&gt; | Network to use (localnet, testnet-asimov) | No |  |
|  | --rpc &lt;rpcUrl&gt; | RPC URL for the network | No |  |
| -h | --help | display help for command | No |  |
