title: update ollama

Manage Ollama models (update or remove)

### Usage

`$ genlayer update ollama [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --model [model-name]  Specify the model to update or remove (default: "") | No |  |
|  | --remove | Remove the specified model instead of updating | No |  |
| -h | --help | display help for command | No |  |
