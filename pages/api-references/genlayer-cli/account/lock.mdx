---
title: account lock
---

Lock account by removing private key from OS keychain

### Usage

`$ genlayer account lock [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --account &lt;name&gt;  Account to lock | No |  |
| -h | --help | display help for command | No |  |
