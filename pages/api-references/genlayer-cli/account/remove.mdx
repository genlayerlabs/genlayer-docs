Remove an account

### Usage

`$ genlayer account remove [options] <name>`

### Arguments

- `<name>`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  | --force | Skip confirmation prompt (default: false) | No |  |
| -h | --help | display help for command | No |  |
