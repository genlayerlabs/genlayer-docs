Create random validators

### Usage

`$ genlayer localnet validators create-random [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --count &lt;count&gt;             Number of validators to create (default: "1") | No |  |
|  |  | --providers &lt;providers...&gt;  Space-separated list of provider names (e.g., | No |  |
|  |  | openai ollama) (default: []) | No |  |
|  |  | --models &lt;models...&gt;        Space-separated list of model names (e.g., gpt-4 | No |  |
|  |  | gpt-4o) (default: []) | No |  |
| -h | --help | display help for command | No |  |
