---
title: localnet validators update
---

Update a validator's details

### Usage

`$ genlayer localnet validators update [options] <validatorAddress>`

### Arguments

- `<validatorAddress>`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --stake &lt;stake&gt;        New stake for the validator | No |  |
|  |  | --provider &lt;provider&gt;  New provider for the validator | No |  |
|  |  | --model &lt;model&gt;        New model for the validator | No |  |
|  |  | --config &lt;config&gt;      New JSON config for the validator | No |  |
| -h | --help | display help for command | No |  |
