---
title: localnet validators create
---

Create a new validator

### Usage

`$ genlayer localnet validators create [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  | --stake &lt;stake&gt; | Stake amount for the validator (default: 1) (default: "1") | No |  |
|  | --config &lt;config&gt; | Optional JSON configuration for the validator (e.g., '\{"max_tokens": 500, "temperature": 0.75\}') | No |  |
|  | --provider &lt;provider&gt; | Specify the provider for the validator | No |  |
|  | --model &lt;model&gt; | Specify the model for the validator | No |  |
| -h | --help | display help for command | No |  |
