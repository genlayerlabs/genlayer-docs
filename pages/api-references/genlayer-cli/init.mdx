---
title: init
---

Initialize the GenLayer Environment

### Usage

`$ genlayer init [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  |  | --numValidators &lt;numValidators&gt;       Number of validators (default: "5") | No |  |
|  | --headless | Headless mode (default: false) | No |  |
|  | --reset-db | Reset Database (default: false) | No |  |
|  |  | --localnet-version &lt;localnetVersion&gt;  Select a specific localnet version | No |  |
|  |  | (minimum: v0.65.0) (default: "v0.65.0") | No |  |
|  | --ollama | Enable Ollama container (default: false) | No |  |
| -h | --help | display help for command | No |  |
