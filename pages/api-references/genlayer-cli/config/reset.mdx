---
title: config reset
---

Reset a configuration value to its default
Arguments:
key         Configuration key to reset

### Usage

`$ genlayer config reset [options] <key>`

### Arguments

- `<key>`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
| -h | --help | display help for command | No |  |
