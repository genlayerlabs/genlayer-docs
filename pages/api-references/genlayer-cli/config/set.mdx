Set a configuration value
Arguments:
key=value   Configuration key-value pair to set

### Usage

`$ genlayer config set [options] <key=value>`

### Arguments

- `<key=value>`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
| -h | --help | display help for command | No |  |
